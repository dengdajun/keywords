### 环境
windows对应的设置环境变量即可
```cmd
export GOPROXY=https://goproxy.io
# 可以不开启module，大部分go老模块不支持
export GO111MODULE=on
export all_proxy=https://goproxy.io
上面的方式如果有问题,使用git-bash执行golang.sh来解决
```
